Voici la documentation pour la version sass du framework DBSASS.

Grille

La grille est une grille responsive composée de 12 colones (modifiable). Sur des ecrans de tablette, les colonnes se redimensionneront soit à 50% soit à 100% en fonction de la construction choisie. Il est également possible de sauté des colones grace à des offsets. Vous pouvez redimenssionez votre navigateur web pour voir le comportement des colones.

Importation du mixin

  
  @import "/chemin/vers/le/mixin/_grid.scss";

Construction Sass

Il va d'abord falloir construire la rangée à l'aide du code suivant :

  
  .row{
    @include row;
  }

Ensuite nous passons à la création des colonnes :

  
  .row{
    @include row;

    .column{
      @include col($largeur, $tablette, $total, $ecart);
    }
  }
Variable Description Valeur défaut Valeurs possibles
$largeur Nombre de colonne aloué à l'élément 3 Nombre de 1 à $total
$tablette Largeur de la colonne sur tablette "l" "l" (100% largeur) / "m" (50% largeur)
$total Nombre total de colonne 12 de 0 à +inf
$ecart Padding entre chaque colonne 1em Toutes valeurs

Vous pouvez également rajouter des offsets (saut de colonne):

  
  .column{
    @include col($largeur, $mobile, $total, $ecart);
    @include offsetleft($nombre_colonnes); // saut de colonne à la gauche de l'élement
    @include offsetright($nombre_colonnes); // saut de colonne à la droite de l'élement
  }
Variable Description Valeur défaut Valeurs possibles
$nombre_colonnes Nombre de colonne(s) sautée(s) 0 Nombre de 0 à $total

Fil d'ariane

Le fil d'ariane est un outil de navigation pour l'utilisateur. Ca lui permet de se repérer parmis les pages par lesquelles il est passé pour en arrivé la.

Importation du mixin

  
  @import "/chemin/vers/le/mixin/_breadcrumb.scss";

Construction Sass

  
  .breadcrumb {
    @include breadcrumb($couleur, $survol, $temps);
  }
Variable Description Valeur défaut Valeurs possibles
$couleur Couleur du texte Black Une couleur
$survol Couleur lors du survol d'un élement #ef473a Une couleur
$temps Durée de l'animation du survol .35s Une durée

Boutons

Voici les boutons du framework.

Importation du mixin

  
  @import "/chemin/vers/le/mixin/_btn.scss";

Construction Sass

Il faut d'abord construire le bouton :

  
  .btn {
    button {
      @include btnstruct($taille, $hauteur, $largeur, $padding);
    }
  }
Variable Description Valeur défaut Valeurs possibles
$taille Taille de la police 13px Une taille
$hauteur Hauteur minimale de la ligne 1.25 Un nombre
$largeur Epaisseur de la police 400 Un nombre
$padding Padding du bouton 0.5em 1em Une valeur

Maintenant nous pouvons commencer à styliser le bouton : 

  
  .btn {
    button {
      @include btnstruct($taille, $hauteur, $largeur, $padding);
    }

    .btn-style {
      @include btncolor($couleur, $temps);
    }
  }
Variable Description Valeur défaut Valeurs possibles
$couleur Couleur du bouton #4183D7 Une couleur
$temps Durée de l'animation au survol 0.35s Une durée

Vous pouvez également decider si le bouton est désactivé : 

  
  .btn {
    button {
      @include btnstruct($taille, $hauteur, $largeur, $padding);
    }

    .btn-style {
      @include btncolor($couleur, $temps);
    }

    .desactive {
      @include disabled;
    }
  }
